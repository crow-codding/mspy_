## ✅ 已解决的问题

### 1. 正确的分组机制

```python
# 从 NAME 提取 INCHIKEY（27 字符格式）
def extract_inchikey_from_name(name_series):
    inchikey_pattern = r'([A-Z0-9\-]{27})'
    return name_series.str.extract(inchikey_pattern, expand=False)

df['INCHIKEY'] = extract_inchikey_from_name(df['NAME'])
```

**解决效果**：确保同一化合物的谱图被正确归为一组

### 2. 容差参数优化

PPM_TOLERANCE = 30.0  # 从 20.0 提高到 30.0

## 💡 关键发现

### Bonanza 算法原理验证

```
Bonanza = Σ√(intensity1 × intensity2) / √(Σintensity1² × Σintensity2²)
```

| 机制 | 说明 |
|------|------|
| **峰匹配** | 使用 m/z 容差匹配峰（缺失峰设为 0） |
| **归一化** | 消除强度差异（归一化到 0-100） |
| **相似度计算** | 基于匹配峰的强度乘积开方求和 |

### 合理相似度范围

| 情况 | 相似度范围 | 说明 |
|------|------------|------|
| 同一化合物重复实验 | 0.85-0.95 | 高质量匹配 |
| 同一化合物不同条件 | 0.60-0.85 | 有合理差异 |
| 不同化合物 | <0.40 | 通常低于 0.5 |
